# AgentClinic Tech Stack

AgentClinic is built on a server-side TypeScript stack with a focus on reliability and broad familiarity.

- Runtime: Node.js
- Framework: Express
- Database: SQLite
- Language: TypeScript
- Browser support: modern browsers with responsive design
- UI framework: Pico CSS for a lightweight, responsive design system with an orange-and-black brand palette
- Responsive design: UI should adapt gracefully to mobile and desktop viewports
- Validation: Vitest for test-driven validation and component-level checks
