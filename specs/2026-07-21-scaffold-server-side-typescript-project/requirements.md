# Requirements: Scaffold Server-Side TypeScript Project

## Scope
- Establish the backend foundation for AgentClinic using Node.js, TypeScript, Express, and SQLite.
- Create the initial repository layout and developer tooling.
- Ensure the app can start locally and connect to a SQLite database.

## Decisions
- Use server-side TypeScript to match stakeholder demand for a popular TypeScript stack.
- Choose Express for a widely familiar framework and easy onboarding.
- Use SQLite as the persistence engine for a lightweight, local-first experience.
- Keep the first phase focused on setup and verification rather than business features.

## Context
- The mission prioritizes agent relief, reliability, and an approachable learning/demo experience.
- The tech stack confirms Node.js, TypeScript, Express, and SQLite as the implementation basis.
- This scaffold will support future development of agents, ailments, therapies, and appointments.
