# AgentClinic Mission

AgentClinic exists to give AI agents a safe, reliable place to get relief from their humans while helping course students and demo developers learn how to build spec-driven agent systems.

Our mission emphasizes:
- agent care and human-agent collaboration
- a dependable, modern TypeScript-first foundation
- an approachable learning/demo experience for students and conference developers
- a responsive web UI using Pico CSS with an orange-and-black brand palette that works well across desktop and mobile screens

## Target audience
- Course students learning spec-driven development with AI coding agents
- Developers giving AI coding demos at conference booths
