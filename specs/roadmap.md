# AgentClinic Roadmap

This roadmap is split into very small phases to support iterative delivery.

## Phase 1 - MVP foundation
- scaffold server-side TypeScript project
- add Express-based API and SQLite persistence
- design core data models: agents, ailments, therapies, appointments
- build a basic dashboard UI for agents and staff
- **Phase 1 complete**

## Phase 2 - Agent and appointment workflows
- implement agent record management
- implement ailment and therapy management
- add appointment booking and status updates

## Phase 3 - polish and marketing readiness
- refine UI for attractiveness and usability
- improve browser compatibility for modern browsers
- add onboarding text and demo-friendly examples
- prepare stakeholder-facing documentation and launch notes
